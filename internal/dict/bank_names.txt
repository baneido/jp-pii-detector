# 全国銀行協会（Zengin）加盟金融機関の名称のうち、著名な銀行・信用金庫・労働金庫
# を手作業で収録した「代表サブセット」。銀行名は正規化済み（全角英数字は半角折り畳み
# 済み）の完全な名称（本体＋業態サフィックス）で 1 行 1 件。
#
# 重要な注意（internal/dict/bank_names.go・docs/development.md も参照）:
#   - これは zengin-code 等の公式マスタ（全国約 1,100 行・支店を除く）ではなく、
#     著名な金融機関のみを一般常識の範囲で手作業収録した代表サブセットである。
#     全国のすべての地方銀行・信用金庫・信用組合・JAバンクを網羅しない。
#   - 金融機関の合併・改称は頻繁に起こるため、このリストは経年で古くなりうる。
#     取り込み前にメンテナが実データ（zengin-code 由来等、ライセンス確認済みのもの）
#     との突合・サインオフを行うこと（Issue #61 参照）。
#   - jp-bank-account ルールの ContextPatterns（正規表現アンカー
#     "(候補)(銀行|信用金庫|信用組合|信金|信組|労働金庫|ろうきん|農協)"）から
#     切り出された候補文字列の接尾部分を長い順にこの辞書で O(1) 検証する。
#
